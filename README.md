# Rubrics Check 质检任务

1. 根据prompt检查rubrics是否正确。
2. 检查每条repo的链接网页是否完成了rubrics里的每一条要求并给出1，0的打分。
3. 质检人员给出质检备注，帮助标注人员定位问题。

# 原始质检作业流程痛点
1. 表格作业需要一条一条点开看，效率低下。
2. 每个链接网页需要手动打开，跳转到新页面去操作，然后再返回来填写。
3. 评分二维数组不易于人类阅读，每个评分不能快速定位到具体是哪一个链接的哪一条rubrics。
4. 质检备注需要手动填写，容易遗漏或不够详细。
